```
'번역중:아무나'' - (아무나)가 번역 중, 완료 후 '번역완료'
'1차리뷰중:아무거'' - (아무거)가 1차 리뷰 중, 완료 후 '1차리뷰완료'
'2차리뷰중:아무개'' - (아무개)가 2차 리뷰 중, 완료 후 '2차리뷰완료'
'n차리뷰중:아무고' - 3차, 4차 등 뭔가 찜찜하면 계속
'일단완료' - 일단 번역과 리뷰가 끝난 상태
```
